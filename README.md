# Snittpris-NO4-Produksjonstimer
Verktøyet beregner et ideelt case der kraftverket produserer i de høyest prisede timene i prisområdet NO4. Brukeren kan justere antall fullasttimer og se hvordan redusert driftstid, for eksempel fra 4000 til 2000 timer, påvirker gjennomsnittlig kraftpris og hvor mye inntjeningen potensielt kan øke.
